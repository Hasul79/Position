Դիրքի position տարբեր արժեքներ

position: static

Բոլոր վանդակների համար position լռելյայն / default արժեքը:

static

position: relative

Տեղադրված (կանաչ) վանդակն ունի վերևի և ձախից՝ 32px օֆսեթ արժեքները, և այդպիսով իր նորմալ դիրքից վերևից և ձախից շեղվում է 2em-ով:

relative

position: absolute

Տեղադրված (կանաչ) վանդակը բացարձակապես տեղադրված է աջ օֆսեթ արժեքով. right: 0։ Սա տուփը մղում է տարայի աջ եզրին, բայց պահպանում է իր ուղղահայաց դիրքը:

absolute

position: fixed

Տեղադրված (կանաչ) վանդակն անում է իր գործը: right: 0 and top: 0 օֆսեթ արժեքներով այն միշտ կլինի ամբողջ էջի վերևի աջ անկյունում:

fixed

position: sticky (top)

Տեղադրված (կանաչ) տուփն իրեն պահում է համեմատաբար տեղադրված տուփի նման, քանի դեռ դրա պարունակող բլոկը չի պտտվել նորմալ հոսքի վերևի դիրքը position “out-of-frame / շրջանակից դուրս”.

Այնուհետև վանդակը կպահպանի վերին դիրքը տեսանելի տարածքի համեմատ, քանի որ դրա տակ գտնվող բովանդակությունը ոլորվում է դեպի տեսադաշտ:

sticky (top)

position: sticky (bottom)

Տեղադրված (կանաչ) տուփն իրեն պահում է համեմատաբար տեղադրված տուփի նման, քանի դեռ դրա պարունակող բլոկը չի պտտվել նորմալ հոսքի ստորին դիրքը position “out-of-frame / «շրջանակից դուրս»”.

Այնուհետև վանդակը կպահպանի ներքևի դիրքը տեսանելի տարածքի համեմատ, քանի որ դրա վերևում գտնվող բովանդակությունը ոլորվում է դեպի տեսադաշտ

position: sticky (left)

Տեղադրված (կանաչ) տուփն իրեն պահում է համեմատաբար տեղադրված տուփի պես, քանի դեռ իր պարունակող բլոկը չի պտտվել նորմալ հոսքի : ձախ դիրքում՝ position “out-of-frame” / «շրջանակից դուրս».

Այնուհետև վանդակը կպահպանի ձախ դիրքը տեսանելի տարածքի համեմատ, քանի որ աջ կողմի բովանդակությունը դիտվում է:
